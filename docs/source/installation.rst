Installation
------------
To use Epitools, first install it using pip:

.. code-block:: console

   $ pip install epitools
   
Then import it in python console

>>> import epitools
>>> import epitools.epigrowthmodel as egm
>>> from epitools import tcoeff.get_tcoeff as tc
